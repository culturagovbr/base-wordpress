Adiciona licenças Creative Commons, Copyright e domínio público para cada post individualmente.

== Description ==

Este plugin para wordpress foi criado com a intenção de divulgar as licenças Creative Commons, Copyright e domínio público para conteúdos, nele você decide de que forma seu conteúdo pode ser usado por outras pessoas, você pode escolher se:

1. A obra ficará livre para ser distribuída sem fins comerciais;
2. Apenas o autor terá autonomia para ceder ou comercializar a obra;
3. Qualquer pessoa poderá copiar e distribuir a obra, desde que atribuam o crédito da mesma.

Este modelo de cadastro de licenças foi feito baseado no cadastro de conteúdos do projeto iTEIA -  http://www.iteia.org.br/ um projeto do Instituto Intercidadania - http://www.intercidadania.org.br/

== Installation ==

Como instalar:

1. Envie a pasta com os arquivos do plugin para dentro de wp-content/plugins.
2. Vá até o menu Plugins, procure pelo plugin WP-licenses e clique em Ativar.
3. Vá até o menu 'configurações > wp-licenses' e configure sua licença.
